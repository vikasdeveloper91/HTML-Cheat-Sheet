Special Cheat Sheet of Some HTML Tags !!!
  1. HTML Boiler Plate : Simple HTML Boiler Plate 
    
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
  2. Anchor tag : Creates link 
    <a href="https://www.google.com">Visit www.google.com</a>
  3. img tag : To insert image in document 
    <img src=" " alt=" " width=" " height=" ">
  4. Table tag : Creates a table in document 
    
<table>
<tr>
    <th>Col-1 Header</th>
    <th>Col-2 Header</th>
</tr>
<tr>
    <td>Data1</td>
    <td>Data2</td>
</tr>
</table>
  5. ul tag : To create unordered list in document 
    
<ul>
<li>Data 1</li>
<li>Data 2</li>
<li>Data 3</li>
</ul>
  6. ol tag : To create ordered list in document 
    
<ol type="I">
<li>Data 1</li>
<li>Data 2</li>
<li>Data 3</li>
</ol>
  7. p tag : It defines paragraph in document 
    <p>This is some text in a paragraph.</p>
  8. h1--h6 tag : To create headings in document 
    
<h1>This is most important heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is least important heading 6</h6>
  9. form tag : To create form with input fields 
    
<form action="/" method="">
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname"><br><br>
<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname"><br><br>
<input type="submit" value="Submit">
</form>
  10. button tag : To define clickable button 
    <button> type="button">Click Me!</button>

I hope you liked this cheat sheet containing some HTML important tags. I know, there are only few tags, but will be more helpful for you.
